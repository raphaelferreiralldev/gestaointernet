## Modulos

- **Dashboard** - KPIs, graficos de receita, crescimento de clientes e consumo de rede
- **Clientes** - Cadastro, busca, filtros, ativacao e suspensao de assinantes
- **Planos** - Gestao de planos de internet com velocidade e precificacao
- **Financeiro** - Emissao e controle de faturas, registro de pagamentos
- **Rede** - Monitoramento de conexoes, latencia, uptime e eventos de rede
- **Chamados** - Abertura e gestao de tickets de suporte com prioridades

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS
- Recharts
- React Router v6
